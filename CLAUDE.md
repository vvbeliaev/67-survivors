# 67-survivors — Design Doc v0.1

## Концепт

Кооп-рогалик на 2–4 игрока, вид сверху. Гибрид: классовая RPG-боёвка (Magicka / Hammerwatch) + билд-строение через стакающиеся апгрейды (Vampire Survivors / Айзек). Пати запирается на арене, волны давят, разойтись нельзя, оббежать нельзя. Цель — пережить N минут и завалить финального босса. Плохая игра ИЛИ невезучий рандом апгрейдов = вайп = рестарт с нуля.

**Хук**: без скоординированной пати ты ничего не сделаешь. Каждый класс закрывает уникальную роль; провал одного игрока — провал всех.

## Сессия

- Длительность забега: 20–30 минут (target, уточнить на прототипе).
- 1 арена в MVP. Тиры локаций и переходы — Tier 2.
- Пати-онли по умолчанию. Соло — TBD.
- Дублирование классов — по умолчанию запрещено.

## Камера и пространство

Единая камера, привязана к центроиду пати, с динамическим зумом. Расходиться нельзя — экран сжимает. Это база геймплея: волну нельзя оббежать, нужно держать фронт. Стан одного игрока — позиционная проблема всей пати (от него нельзя отъехать, надо защищать).

## Управление

- **WASD** — передвижение.
- **ЛКМ** — основной активный скилл (привязан к курсору).
- **ПКМ** — второй активный скилл (привязан к курсору).
- **Space** — класс-специфичная утилка/мобильность: рывок, телепорт, землетрясение, агро-пульс — зависит от класса.
- **Авто-атака** оружием — постоянно, в стиле VS.

## Боевая модель

У каждого класса фиксированное оружие, стреляющее автоматически (VS-стайл). Поверх — два активных скилла (ЛКМ/ПКМ), требующих прицеливания и осмысленного нажатия. Активный скилл-уровень нагрузки = ближе к Magicka/Hammerwatch, чем к чистому VS.

## Прогрессия

### Внутри забега

**Уровни пати.** Сундук поднимается одним игроком, но повышает уровень всей пати одновременно. Уровней «на одного» нет — это удерживает пати в общем темпе.

**Апгрейды.** Каждому игроку индивидуальный выбор из 3 случайных апгрейдов на левел-ап (как в VS). **Пул общий между классами** — числовые: ХП, скорость, кулдауны, рейндж, лайфстил, реген маны, скорость каста и т.д. Апгрейды стакаются.

**Дерево класса.** Маленькое (5–8 узлов), числовые баффы, специфичные для кор-механики класса (например, у Мага — ёмкость маны, реген, скорость каста).

**Архетип.** В определённый момент забега игрок осознанно выбирает архетип (сабкласс), фиксирующий направление билда. Например, Маг → Тайный / Стихийный / Призыватель. После выбора пул апгрейдов и дерево интерпретируются под архетип. Архетип **не эмерджентный** — это явный чекпойнт.

### Между забегами

TBD. Дефолтная позиция: минимальная мета-прогрессия (косметика + анлоки классов / новых апгрейдов в пул). Перманентные числовые баффы — **нет**, чтобы не убивать рестарт-чувство.

## Кооп-дизайн

Кооп держится на **жёстком разделении ролей**, а не на синергиях апгрейдов:

| Класс      | Роль                            |
| ---------- | ------------------------------- |
| Берсерк    | Tank / агро / контроль волны    |
| Бард       | Support / хил / групповые баффы |
| Маг        | AoE / контроль волн             |
| Арбалетчик | Single target / элитки и боссы  |

Без любой роли пати не вытягивает. Бард фейлит → никого не лечат. Берсерк не держит агро → фронт рассыпается. И т.д.

Дополнительный слой — **небольшие ауры близости** (классо-специфичные пассивы), мотивирующие пати держаться рядом и усиливающие соседей. Кросс-классовых апгрейд-синергий в MVP нет — это Tier 2.

## Классы

### Берсерк — tank / контроль волны

- **Оружие**: ближний бой, AoE-крутилка вокруг себя.
- **ЛКМ**: рывок-удар, агрит врагов в линии.
- **ПКМ**: рев — тянет агро в радиусе, влияет на глобальное движение волны.
- **Space**: землетрясение — AoE-стан вокруг себя (не мобильность).
- **Кор-механика**: получение урона = ресурс. Часть скиллов скейлится от потерянного ХП.
- **Архетипы (TBD)**: Защитник (танк/щит) / Кровавый (lifesteal/risk) / Командир (агро/контроль).

### Маг — AoE / контроль волн

- **Оружие**: автокаст снарядов.
- **ЛКМ / ПКМ**: тяжёлые скиллы с большим уроном и рейнджем, расходуют ману.
- **Space**: блинк (короткий телепорт по курсору).
- **Кор-механика**: мана. Прокачка влияет на ёмкость, реген, стоимость и агрессивность трат.
- **Архетипы (TBD)**: Тайный (долгие заряженные удары — высокий риск/высокий урон) / Стихийный (быстрые AoE) / Призыватель (миньоны).

### Бард — support / хил

- **Оружие**: слабый снаряд для самозащиты.
- **ЛКМ / ПКМ**: групповые баффы (хил, скорость, броня), активируются через QTE-микро-механику. Успешный QTE = бафф пати; провал = **просто нет баффа** (никаких malus’ов на пати).
- **Space**: дэш-уворот в безопасную сторону.
- **Кор-механика**: единственный хилер. Тонкий — критично не получать урон, позиционирование = всё.
- **Архетипы (TBD)**: Хилер / Бафер / Дебафер врагов.

### Арбалетчик — single target

- **Оружие**: арбалет с зарядкой. Дольше заряжаешь → больше урон, но во время зарядки сильно замедлен.
- **ЛКМ**: выстрел.
- **ПКМ**: бронебойный/особый болт с длинным кулдауном.
- **Space**: перекат (i-frames на короткую дистанцию).
- **Кор-механика**: баланс «зарядка vs мобильность». Главный убийца элиток и боссов.
- **Архетипы (TBD)**: Снайпер (дальность/крит) / Охотник (мобильность/скорострельность) / Капканщик (контроль/ловушки).

## Враги и волны

В MVP — пул базовых архетипов (рашер, танк, дальний, элитка). Сложность скейлится во времени по VS-модели: количество, миксы, элитки, мини-боссы каждые ~5 минут, финальный босс на N+1 минуте. Детальный дизайн волн — отдельный документ.

## Смерть и вайп

- Игрок умер — лежит, **автореспавн через 30 секунд**, если пати жива.
- Все легли одновременно — забег окончен, рестарт.
- Активное поднятие союзника (Left 4 Dead-стайл) vs автореспавн — TBD. Активное добавляет тактику и драму, но рискует не работать в плотных волнах.

## Win condition

Пережить таймер и завалить финального босса. Всё остальное — недоделанный забег.

## Tier 2 (после MVP)

- Несколько арен и переходы между ними.
- Кросс-классовые апгрейд-синергии (Маг + Арбалетчик: холод → картечь и т.п.).
- Расширение пула апгрейдов и архетипов.
- Дополнительные классы.
- Возможно: соло-режим с балансом под одного.
